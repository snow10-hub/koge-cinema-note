# KOGE CINEMA NOTE

映画を検索し、気になる作品を保存し、観た映画の記録を残せる映画管理サイトです。

TMDB APIを利用し、映画検索、今週の話題作表示、映画詳細表示、My List保存、レビュー投稿、ジャンル検索、関連映画表示などを実装しました。

単なる映画検索サイトではなく、「映画を探す」「観たい作品を保存する」「観た記憶を残す」「次の作品へつなげる」体験を意識して制作しています。

## 工夫した点

### 映画を探して終わりにしない体験設計

検索機能だけでなく、My List保存、レビュー投稿、関連映画表示を追加しました。

映画を探すだけで終わらせず、気になる作品を保存し、観た映画の感想を残し、次に観たい作品へつなげられる流れを意識しています。

### コンポーネント単位で管理しやすい構成

Header、Hero、MovieCard、MovieList、FavoriteButton、ReviewSectionなど、役割ごとにコンポーネントを分けました。

画面ごとの役割を整理することで、UI調整や機能追加の影響範囲を把握しやすい構成にしています。

### 外部APIのデータを自然に見せる調整

TMDB APIから取得した映画データを、そのまま表示するのではなく、表示用に整形しています。

ジャンル名の一部を自然な日本語表記に変換し、画像やあらすじが取得できない場合にも表示が崩れないように調整しました。

また、検索結果が見つからない場合には、英題や正式タイトルでの再検索を想定した表示にするなど、外部APIの仕様に依存する部分も考慮しています。

### 映画サイトらしいUIデザイン

ダークトーンをベースに、sky系のアクセントカラーを使用しました。

ヒーローエリアでは背景画像の上にグラデーションや半透明の背景を重ね、映画館のような雰囲気と文字の読みやすさを両立させています。

スマートフォンでは、一般的な三本線メニューではなく、夜の映画館をイメージした三日月アイコンのメニューを採用しました。

### localStorageを使った保存機能

My Listやレビュー情報はlocalStorageに保存しています。

バックエンドや認証を使わない構成でも、ユーザーが映画を保存し、レビューを残せるように実装しました。
